# Personal documentation

## MISC

- settings (might need root level):
    - `minicom -s`
- open specific port:
    - `minicom -D /dev/ttyUSB0`
- read from the port
    - `sudo cat /dev/ttySx`
- write to the port
    - `echo "something" | tee /dev/ttySx`
